# Prompt Library

Available prompts:
- `docs/prompts/UX_UI_MASTER_PLAN_PROMPT.md` — Regenerate the UX/UI master plan, including repo audit, screen contracts, and roadmap.
- `docs/prompts/CODEX_FULL_APP_E2E_QA_SECURITY.md` — Full-app E2E validation, QA debug, security audit, and fix-to-green workflow.
- `docs/prompts/ui_ux_modernization_expert.md` — AI Persona for Executive UI/UX Architecture, focusing on design systems and perceived performance.
- `docs/prompts/react_auto_save_refactor.md` — Guide for refactoring forms to modern "Auto-Save" patterns using Server Actions.
- `docs/prompts/critique_to_code_workflow.md` — Workflow for converting loose critique/feedback into systematic implementation steps.
- `docs/prompts/documentation_and_handoff_specialist.md` — Persona for generating high-quality documentation and handoff logs.

How to use:
1. Open Codex CLI in the RitualFin repo.
2. Copy/paste the prompt from the chosen file and run.
